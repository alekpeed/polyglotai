# Android (placeholder)

`packages/` — `core-learning`, `spaced-repetition`, `ai-orchestration`, `language-pack-sdk`,
and `shared-types` — behind a native Android shell and its own SQLite storage layer. See
`docs/mvp-implementation-plan.md` for the architecture that makes this reuse possible.
